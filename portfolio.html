Gerador de Frases Aleatórias de Mogli, o Menino Lobo (HTML, CSS e JavaScript)

Clique aqui para ver o site!

Neste projeto, desenvolvi uma página interativa que gera frases aleatórias do clássico desenho "Mogli, o Menino Lobo". A página foi criada utilizando JavaScript e inclui funcionalidades dinâmicas para melhorar a experiência do usuário.

Funcionalidades:

  • Gerador de Frases Aleatórias: A cada clique, uma nova frase do desenho é gerada, proporcionando uma experiência nostálgica e divertida.
  • Contador de Frases: Um contador é atualizado a cada nova frase gerada, permitindo ao usuário ver quantas frases foram exibidas até o momento.
  • Contador de Palavras Específicas: Adicionalmente, implementei um contador dedicado que rastreia todas as vezes que a palavra "abutres" aparece nas frases, adicionando um elemento de curiosidade ao projeto.
  • Botão de Reset: Para maior controle, há também um botão de reset que permite ao usuário redefinir ambos os contadores e iniciar uma nova sessão.
